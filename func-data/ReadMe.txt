This file contains six noisy function datasets used for testing the DNN-Solver. 

Each text file consists of 201 data points, with the first column representing 'x' and the second column representing 'y'.

These data points are provided in 
(Gálvez, Akemi, and Andrés Iglesias. "Efficient particle swarm optimization approach for data fitting with free knot B-splines." Computer-Aided Design 43.12 (2011): 1683-1692.) 
(Yoshimoto, Fujiichi, Toshinobu Harada, and Yoshihide Yoshimoto. "Data fitting with a spline using a real-coded genetic algorithm." Computer-Aided Design 35.8 (2003): 751-760.)
